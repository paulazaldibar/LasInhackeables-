PRÁCTICA 5 — WEB SCRAPING

Reflexión

En este apartado se ha desarrollado un script en Python utilizando requests y BeautifulSoup para hacer web scraping sobre la página de Hola.com.

Se ha conseguido extraer y guardar el texto de la web (títulos y párrafos), descargar imágenes, recopilar enlaces, buscar datos de contacto como emails y teléfonos, y descargar posibles archivos como PDFs.

En algunos casos, como los emails, no se han encontrado resultados, y en los teléfonos pueden aparecer valores que no son reales, debido a cómo está estructurada la página.

En general, se ha logrado recopilar información de la web de forma automática y almacenarla en distintos archivos.

2.a. ¿Qué se puede conseguir con este tipo de programas?

Con este tipo de programas se puede recopilar información de páginas web de forma automática, como textos, imágenes, enlaces, datos de contacto o archivos. Esto permite obtener grandes cantidades de datos sin tener que hacerlo manualmente.

Un uso real sería analizar varias páginas web de noticias (por ejemplo, Hola.com) para recopilar titulares y ver cuáles son los temas más frecuentes en un periodo de tiempo.

2.b. OSINT: ¿Qué información se puede scrapear?

En el contexto de OSINT (inteligencia de fuentes abiertas), se puede recopilar de forma masiva información pública como:

  • Nombres y datos personales visibles
  • Correos electrónicos
  • Teléfonos
  • Perfiles en redes sociales
  • Noticias o menciones en medios
  • Imágenes y contenido publicado
  • Enlaces a otras páginas relacionadas

Todo esto permite construir un perfil bastante completo de una persona o empresa usando solo información pública.

2.c. Redes sociales: utilidad y problemas

Sí, el scraping es útil en redes sociales para analizar tendencias, opiniones o comportamiento de usuarios.

Sin embargo, presenta varios problemas:

  • Muchas plataformas bloquean el scraping o lo limitan
  • Uso de JavaScript dinámico (más difícil de extraer)
  • Problemas legales y de privacidad
  • Posibles bloqueos de IP o cuentas
  • Cambios frecuentes en la estructura de la web
