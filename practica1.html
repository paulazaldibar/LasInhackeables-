PRÁCTICA 1 - Mei Torres, Lara Poncela, Paula de Velasco

No hemos podido completar la práctica ya que tuvimos varios problemas más importantes. Empezamos descargando Rufus, y en el siguiente paso, que es descargarse Ubuntu Desktop, nos equivocamos fiándonos del peso del archivo y por error nos descargamos Ubuntu Server. Una vez descargado “Ubuntu” completamos el paso de añadir Ubuntu a Rufus para configurar el USB booteable. Reiniciamos el ordenador y accedimos a la BIOS a la sección de Boot option priorities, en la que pusimos el USB en primer lugar. Guardamos la configuración y reiniciamos el ordenador, ejecutamos “try or install Ubuntu”, donde empezaron las preguntas para configurar el Ubuntu Server. Pensando que solamente había que darle a siguiente, avanzamos unas 3 ventanas de configuración, entre ellas preguntas relacionadas con la conexión web…

Al llegar a una pantalla en la que se pedía el nombre, usuario, contraseña, root, etc, nos alarmamos ya que no nos sonaba el haber comentado este paso en ningún momento. En consecuencia, Paula perdió Windows y todo lo que contenía por nuestro error ya que Ubuntu Server es para profesionales y se usa en una terminal y no tiene nada que ver con Ubuntu Desktop, que es un sistema operativo y que además tiene una opción de probarlo sin instalar ni tocar archivos del disco duro de Windows.

En conclusión, hemos aprendido una lección muy importante, deberíamos haberlo hecho con más calma, leyendo lo que estábamos descargando y no darle a “siguiente” sin saber o entender lo que aparecía.
